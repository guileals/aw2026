# Desafio de Programação: Criando um Mini-Site Pessoal

Objetivo:

Desenvolver uma página web simples utilizando HTML, seguindo as boas práticas de programação.

**Questões principais:**

1. **Estrutura do HTML:**
   - Criar um arquivo "desafio1.html"
   - Criar uma estrutura básica de página HTML5 com tags semânticas como `<header>`, `<main>`, e `<footer>`.
   - Definir a linguagem de programação como sendo português (`lang="pt-BR"`).

2. **Cabeçalho:**
   - Incluir um título (`<h1>`) com o nome do usuário.
   - Incluir um título (`<h2>`) com seu curso.
   - Adicionar um menu de navegação (`<nav>`) com links para "Início", "Sobre mim", "Github" (link para seu repositório pessoal).

3. **Conteúdo Principal:**
   - Desenvolver 4 seções (`<section>`) contendo título, imagem e texto descritivo, incluindo a formatação de elementos como negrito (`<strong>`) e itálico (`<em>`).
   - As imagens devem estar dentro de contâineres genéricos (`<div>`).

4. **Links Internos:**
   - Criar arquivos HTML para a páginas "sobre.html", sendo que a página de "sobre" deve apenas conter também uma estrutura semântica (header, main, footer).

5. **Rodapé**
   - Incluir um título (`<h3>`) com sua instituição de ensino (IFRN - Campus Macau)
   - Criar um texto falando da disciplina Autoria Web
